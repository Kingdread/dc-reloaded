Command line commands

h — show this help l [file] — load a file
a [file] — assemble a file
r — start the execution
c — clear and reset the simulator
pc cell — set the program counter to the given address
g cell — set the program counter and start the program
d delay — set a new delay
e [file] — open the editor
b address — set a breakpoint at the given address
togglegui — enable/disable visualization
update — update the screen
hardcore — set a very small delay
quit — quit dc reloaded

For a more detailed help, click the toolbar button "Open documentation" in the main DC window to open the online documentation.
